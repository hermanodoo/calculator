
# Object Oriented Calculator

## Objects Needed

###1. Menu
	- let's user choose either
		(1) View History
		(2) Calculation
		(3) Exit Program

###2. Operation
	- executes calculation
	- allows:
		(1) addition
		(2) subtraction
		(3) multiplication
		(4) division

###3. Database
	- uses ArrayList<String> to store calculation
	- when called getHistory, prints each item in the ArrayList into String

###4. EquationConverter
	- changes string of equation to list
	- check if the equation is valid
	- return numbers list and ooperation list
